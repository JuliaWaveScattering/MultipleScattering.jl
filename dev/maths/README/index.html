The T-matrix and Multiple Scattering

These short notes introduce the T-matrix, which defines how a particle scatters waves, and shows how it used in multiple scattering from many particles.

Discrete Fourier Transform

These notes give details on Green’s functions and Fourier transforms. We mostly focus on using discrete impulse functions.
